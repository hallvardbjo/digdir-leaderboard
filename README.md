# Digdir Leaderboard

A Spring Boot application with a React frontend for tracking and displaying workout leaderboard data.

## Overview

This application displays leaderboard data for different exercises (bench press, deadlift) by reading records from text files. The data is served via REST API endpoints and consumed by a React frontend.
